# Contributing to Project ARES

Welcome to the ARES tactical mesh intelligence community! We're building the future of decentralized coordination.

## Getting Started

1. Fork the repo.
2. Create your feature branch (`git checkout -b feature/amazing-logic`).
3. Commit your changes (`git commit -m 'Add tactical optimization'`).
4. Push to the branch (`git push origin feature/amazing-logic`).
5. Open a Pull Request.

## Coding Standards

- Follow PEP 8 for Python code.
- Ensure all P2P protocols are documented.
- Security-critical code requires double-review.

## Tactical Integrity

All code must be production-ready and optimized for Edge-AI deployment.
